в проекте 2 класса Server и Client
appServer.php запускает сервер.

appClient.php запускает клиентскую программу, на вход которой можно подать какой нибудь текст,
сервер его перевернет и вернет клиенту.

Каких то конкретных требований не было, по тому что должно ходить между клиентом и сервером, поэтому сделал
вот такой простой вариант.
