1. composer
    ```bash
    composer install
    ``` 
2. 
```bash
mv .env.example .env
```
2. Начните принимать сообщения:

    ```bash
    php app.php server
    ```

3. Подключите клиента

    ```bash
    php app.php client
    ```

4. Отпрвляйте сообщения с клиента
5. Чтобы отключится от сервера и закончить общение, отправьте пустую строку
